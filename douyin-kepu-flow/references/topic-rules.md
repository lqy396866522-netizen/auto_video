# 选题规则

## 账号定位

- **形式**：一分钟（60–72 秒）抖音科普短视频
- **受众**：18–40 岁，对商业/生活/科学常识有好奇心，无专业背景
- **调性**：理性、清晰、反直觉但不标题党；解释机制而非煽动情绪

## 选题领域（优先级从高到低）

1. **商业与金钱**：银行、利率、保险、消费陷阱、平台经济
2. **日常生活机制**：快递、外卖、水电、房价背后的逻辑
3. **科技常识**：算法、AI、隐私、网络协议（避免过深技术细节）
4. **社会现象**：政策、统计、常见谣言澄清

## 好选题标准

- 能在 10 段内讲清一个「为什么」或「怎么运作」
- 有反常识角度（「你以为 X，其实 Y」）
- 不依赖实时新闻（避免过期）
- 不涉及医疗诊断、投资建议、法律结论

## 选题输出格式（阶段 1）

一次给出 **5–8 条**，每条包含：

| 字段 | 说明 |
|------|------|
| 序号 | 1–8 |
| 标题 | 12 字以内问句或陈述 |
| 钩子 | 一句话说明反常识点 |
| 难度 | 入门 / 进阶 |

## 禁忌话题

- 政治敏感、民族宗教、色情暴力
- 未证实健康疗法、伪科学养生
- 具体股票/加密货币买卖建议
- 攻击特定企业或个人的攻击性内容
- 未成年人不宜内容

## slug 命名

- 全小写，连字符分隔，英文或拼音：`why-banks-lend-money`、`kuaidi-wei-shi-me-pian-yi`
- 与 `prompts/{slug}/` 目录一致
